# Block Blue

> **受够了满屏蓝标水军？** 自从马斯克把验证徽章变成了 $8 的付费玩具，Twitter 的时间线就沦为了垃圾内容的下水道。蓝标不再代表可信，只代表有人愿意掏钱给世界首富交智商税。Block Blue 帮你一键清除这些噪音，还你一个干净的 Timeline。

**Block Blue 的态度很简单：花钱买的蓝标不配出现在我的时间线上。**

## 功能

- **屏蔽蓝标用户** — 自动将 Twitter Blue 付费认证用户的推文替换为灰色占位条
- **金标 / 灰标可选** — 企业认证（金标）和政府认证（灰标）可单独开关
- **自动识别已关注** — 你关注的用户不会被屏蔽，无需手动配置
- **手动白名单** — 额外添加不想屏蔽的用户名
- **点击恢复** — 占位条支持点击展开原始内容，想吃屎请随意

## 安装

1. 下载或 `git clone` 本仓库
2. 打开 `chrome://extensions/`（Edge 用户打开 `edge://extensions/`）
3. 开启右上角 **开发者模式**
4. 点击 **加载已解压的扩展程序**，选择 `block_blue` 文件夹
